all known issues

taskly 0.1.2 (current release)

  • taskly is a google-integrated product, which is causing some issues on webkit-based browsers like safari with tasks such as loading google-based fonts and connecting to the google api. everything should have backups (such as fonts) but if not please report it using the feedback form.
  • taskly is a beta product and has not been optimized for mobile devices yet. therefore, there will be website rendering issues on some smaller mobile devices.
  • timers are extremely inconsistent on taskly and often don't work. they will be fixed in the next update but are not functional at the time.
  • there is an issue with the feedback form in which the form is submitted before all required fields have been filled out. this will be fixed in the next update.
